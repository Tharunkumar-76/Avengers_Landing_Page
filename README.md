# Avengers Landing Page

A visually stunning landing page featuring Avengers characters with an interactive image slider. This project demonstrates modern web design techniques including glassmorphism, dynamic animations, and responsive layouts.

## Features

-   **Interactive Character Slider**: Navigate through different Avengers characters using "Next" and "Previous" buttons.
-   **Dynamic Animations**: Smooth transitions and entrance animations for character details.
-   **Glassmorphism Effects**: Stylish frosted glass effect on character information cards.
-   **Responsive Design**: Adapts to different screen sizes (though primarily optimized for desktop viewing).
-   **Hover Effects**: Interactive hover states on buttons and cards.

## Technologies Used

-   **HTML5**: For the structural foundation of the page.
-   **CSS3**: For styling, animations, and layout (using Flexbox and absolute positioning).
-   **JavaScript**: For the slider logic and DOM manipulation.

## Usage

1.  Clone or download the repository.
2.  Open the `index.html` file in your preferred web browser.
3.  Use the "Next" and "Previous" buttons at the bottom to cycle through the characters.
4.  Hover over the "See More" buttons to see the hover effects.

## Project Structure

-   `index.html`: The main HTML file.
-   `style.css`: Contains all the styles and animations.
-   `script.js`: Handles the slider navigation logic.
-   `images/`: Directory containing character images.
